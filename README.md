# Health App

## Prerequisites

`NPM`

`Node`

`Typescript`

## Run Test

On Project root directory:

`./src/script/run-test.sh`

Give execute permission if permission denied.

## Build and Run

This will load test the application with more than 1 lakh data.

On Project root directory:

`./src/script/run-build.sh`

Give execute permission if permission denied.
